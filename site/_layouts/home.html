Create real-time apps quickly

One of TBone's core advantages is running on an asynchronous web server. This allows you to build full-duplex APIs, a core feature of real-time apps. Tbone provides classes to build full-duplex REST endpoints on top of Websockets, so you can enjoy a familiar REST-like API without having to handle websocket communication.
